---
title: '( )'
token: ['(', ')']
---

`( )` 用于分组表达式、函数调用参数列表，以及记录字面量。

---

分组表达式，控制计算顺序：

```mira
(1 + 2) * 3 // 先计算 1 + 2，然后乘以 3
```

---

函数声明和调用：

```mira
fn add(a, b) { a + b }
add(1, 2)
```

---

记录字面量：

```mira
let record = (a: 1, b: 2);
let empty = (); // 空记录
```

---

对于省略键名的单元素记录，必须添加逗号以区别于分组表达式。

```mira
let single = (1,);
```
