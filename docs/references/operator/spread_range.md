---
title: '..'
---

`..` 表示闭区间范围，或在数组/记录字面量及函数调用中表示展开。

---

表示数组字面量、切片和循环的范围：

```mira
let arr = [1..3]; // arr = [1, 2, 3]
arr[0..1]         // [1, 2]
```

```mira
for i in 1..3 {
  // i 依次为 1, 2, 3
  debug_print(i);
}
```

---

展开数组/记录或函数调用：

```mira
[1, ..[2, 3]] // [1, 2, 3]
```

```mira
(a: 1, ..(b: 2, c: 3)) // (a: 1, b: 2, c: 3)
```

```mira
// 同名键以最后一个为准
(..(1, 2), ..(3, 4, 5)) // (0: 3, 1: 4, 2: 5)
```

```mira
sum(..[1, 2, 3]) // 等同于 sum(1, 2, 3)
```
