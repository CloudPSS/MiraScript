---
title: ':'
---

`:` 用于记录字面量、条件表达式和插值格式。

---

在记录字面量中，`:` 用于分隔字段名和字段值。

```mira
let r = (x: 1, y: 2);
```

---

在条件表达式中，`:` 用于分隔条件的不同分支。

```mira
let status = score >= 60 ? "及格" : "不及格";
```

---

在插值格式中，`:` 用于指定格式化选项。在指定格式化选项时，必须使用圆括号 `$(expr:format)`。

```mira
debug_print("π 的值约为 $(PI:.2)"); // 输出: π 的值约为 3.14
```
