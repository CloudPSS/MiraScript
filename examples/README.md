# MiraScript 示例集合

本目录提供按主题组织的 MiraScript 示例脚本，覆盖基础语法、控制流、模式匹配、函数式风格与常见算法。

## 示例索引

### 基础示例

- **`01_hello_world.mira`** - 基本的 Hello World 示例，展示字符串输出和不同类型的字符串字面量
- **`02_comments_identifiers.mira`** - 注释语法和标识符命名规则的完整示例
- **`03_literals.mira`** - 所有字面量类型的综合展示：数字、字符串、记录、数组、函数
- **`05_data_types.mira`** - 展示 MiraScript 支持的所有数据类型：number、string、boolean、nil、record、array、function
- **`06_variables.mira`** - 变量声明、可变性、赋值操作和解构赋值的完整示例

### 字符串和插值

- **`10_string.mira`** - 字符串字面量、转义字符、Unicode 支持和逐字字符串的完整示例
- **`11_interpolation.mira`** - 字符串插值语法、嵌套插值、条件插值和复杂表达式插值
- **`12_template.miratpl`** - MiraScript 模板引擎示例，展示在 HTML 中使用插值和控制结构

### 数据结构

- **`15_array.mira`** - 数组字面量、范围构造、数组访问、切片操作和模式匹配
- **`16_records.mira`** - 记录创建、访问、展开、动态键名和模式匹配

### 运算符和表达式

- **`20_operators.mira`** - 算术、比较、逻辑、模式匹配等所有运算符的使用示例
- **`21_extension_calls.mira`** - 扩展调用运算符 `::` 的链式调用示例

### 控制流

- **`25_control_flow.mira`** - if 表达式、match 表达式、for/while/loop 循环的完整示例

### 模式匹配

- **`30_pattern.mira`** - 基础模式匹配示例，包括常量、变量、记录和数组模式
- **`31_advanced_patterns.mira`** - 高级模式匹配，包括范围模式、关系模式、逻辑模式等

### 函数式编程

- **`40_functions.mira`** - 函数声明、函数表达式、高阶函数、闭包和递归的完整示例
- **`41_fib.mira`** - 斐波那契数列的多种实现方式：递归、尾递归、迭代和带缓存

### 安全和错误处理

- **`50_null_safety.mira`** - 空安全访问、空合并操作符、非空断言的使用
- **`55_error_handling.mira`** - 错误处理模式、类型检查、参数验证和安全操作

### 数学和算法

- **`60_math.mira`** - 数学函数、常数、算法实现
- **`70_algorithms.mira`** - 常见算法实现：排序、搜索、数据结构操作
